# ExactWordToJS

Convert Word file to PDF

```dotnetcli
usage: ExactWordToJS [options]
        -s <folder>|<file>   The Source file or Directory
        -d <folder>          The Destination file or Directory
        -v                   Print verbose.
        -h                   Print this file.
```

For Example:

```dotnetcli
.\ExactWordToJS.exe -s C:\test\test.doc -d C:\test\
```
